🐍 Snakes & Ladders
A Python implementation of the classic Snakes & Ladders board game with two interfaces — a graphical GUI built with Tkinter and a terminal-based CLI version.

📋 Features
GUI Version

Visual 10×10 game board with color-coded snake and ladder cells
Animated dice rolling with Unicode dice faces
Player tokens displayed directly on the board
Live scorecards showing each player's current square
Snake and ladder detection with instant feedback
"Play Again" button to restart without closing the app

CLI Version

Supports 2 to 4 players
Custom player names
Press-to-roll turn system
Real-time position updates with snake/ladder notifications


🚀 Getting Started
Prerequisites

Python 3.x
Tkinter (included with most Python installations)
